category_analysis.py View on GitHub ↗

Analyses grading performance broken down by clinical topic category. Merges category labels into the LLM output data, computes per-category means for human and GPT graders, and produces a radar chart, bar chart, donut chart, and a word cloud of grader comments.

Dependencies

pandas · openpyxl · matplotlib · seaborn · wordcloud · nltk

Usage

python Code/category_analysis.py

# Point to a custom data folder and save figures elsewhere
python Code/category_analysis.py --data-dir Data --out-dir Code/output

Outputs (saved to Code/output/)

radar_chart.png          # Per-category HBTotal — human vs GPT
bar_chart.png            # Grouped bar chart by category
donut_chart_plain.png    # Category distribution with mean annotations
donut_chart_color_coded.png
wordcloud.png            # Nouns/adjectives from grader comments
Updated_clustered_LLM_outputs.xlsx
clinician_grading_analysis.py View on GitHub ↗

Full clinician grading analysis across three raters (Dr. H, Dr. B, Dr3) and multiple study rounds. Computes descriptive statistics, intraclass correlation (ICC) between rater pairs and across all raters, Wilcoxon / Pearson / Cohen's Kappa tests between GPT and human responses, grader bias, and generates figures for mean scores, error bars, editing requirements, multi-round progression, and Round 5 LLM grader comparisons.

Dependencies

pandas · openpyxl · matplotlib · seaborn · scipy · scikit-learn · pingouin · statsmodels

Usage

python Code/clinician_grading_analysis.py

# Custom paths
python Code/clinician_grading_analysis.py --data-dir Data --out-dir Code/output

# Skip ICC computation (faster during development)
python Code/clinician_grading_analysis.py --skip-icc

Outputs (saved to Code/output/)
